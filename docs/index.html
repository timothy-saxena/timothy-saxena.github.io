Parking Lot Management System

As part of my association with TakeUForward coaching, I designed and implemented a Parking Lot Management System to gain hands-on experience with Object-Oriented Programming (OOP) principles and Low Level System Design (LLD) concepts in Java.

The project focuses on building a scalable, modular, and extensible design using real-world constraints and industry-relevant design patterns.

The system handles the entry flow by assigning an appropriate parking slot based on the vehicle type when a vehicle arrives, generating a parking ticket, and marking the slot as occupied.

During the exit flow, the user presents the ticket, the system calculates the parking fee using configurable pricing rules, processes the payment, releases the occupied slot, and generates an exit receipt.

Tech Stack: Java · OOP · Low Level Design · Design Patterns

Task Scheduling Using Topological Sort

Implemented a task scheduling system to resolve dependency constraints using Directed Acyclic Graphs (DAGs) and topological sorting.

The project models tasks as graph nodes and dependencies as directed edges, applying depth-first search–based algorithms to determine a valid execution order.

The solution ensures that all prerequisite tasks are completed before dependent tasks, reflecting real-world scheduling and dependency resolution scenarios.

Tech Stack: Java · Graphs · DFS · Topological Sort
